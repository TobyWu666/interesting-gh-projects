# figlet
### 用文字畫出大型 ASCII 藝術標題

**日期：** 2026-05-13
**Repo：** [cmatsuoka/figlet](https://github.com/cmatsuoka/figlet)
**⭐ Stars：** 原始專案已開源維護數十年
**語言：** C + Python（wrapper）
**安裝：** `sudo apt install figlet`

---

## 📖 這是什麼

figlet 是一個**文字轉 ASCII 藝術的大型標題生成器**。你可以輸入任何字串，它會用預先定義的字形模板（.flf 檔案）把每個字元轉換成多行文字構成的大型字母，最終輸出類似這樣的效果：

```
 _   _      _ _         _____     _           _
| | | | ___| | | ___   |_   _|__ | |__  _   _| |
| |_| |/ _ \ | |/ _ \    | |/ _ \| '_ \| | | |
|  _  |  __/ | | (_) |   | | (_) | |_) | |_| |
|_| |_|\___|_|_|\___/    |_|\___/|_.__/ \__, |
                                      |___/
```

說白了就是：**文字 → 大型 ASCII 藝術字**。常用在 terminal 開機畫面、伺服器歡迎訊息、log 標題、email 標題裝飾。

---

## ⚙️ 原理

```
字元資料庫（.flf 格式，字形模板）
    ↓
figlet 主程式（ C 語言）
    ↓
讀取字元對照表 → 組裝輸出
    ↓
終端機 stdout
```

figlet 使用 **`flf`（FIGlet Font）** 格式來定義字形。每個 .flf 檔案包含：
- 標題行（元資料：字體名、作者、方向等）
- 每個可顯示字元（ASCII 33-126）的 5-8 行像素圖

**字形編碼方式：**
每個字元的圖案用「#」表示像素點，用空格表示空白。例如 `A` 的圖案大約是：
```
  @
 @ @
@@@@@
@   @
@   @
```
最終輸出時，figlet 會把所有字元水平拼接，每一個 "layer"（垂直列）一次輸出。

## 🧪 測試結果（豐富版）

### 1. 標準字體（standard）

```bash
$ figlet -f standard "Hello Toby"
 _   _      _ _         _____     _           _
| | | | ___| | | ___   |_   _|__ | |__  _   _| |_
| |_| |/ _ \ | |/ _ \    | |/ _ \| '_ \| | | |
|  _  |  __/ | | (_) |   | | (_) | |_) | |_| |
|_| |_|\___|_|_|\___/    |_|\___/|_.__/ \__, |
                                      |___/
```

評價：經典風格，像海報標題，邊緣筆直，適合正文標題。

---

### 2. 斜體字體（slant）

```bash
$ figlet -f slant "Hello Toby"
    __  __     ____         ______      __
   / / / /__  / / /___     /_  __/___  / /_  __  __
  / /_/ / _ \/ / / __ \     / / / __ \/ __ \/ / / /
 / __  /  __/ / / /_/ /    / / / /_/ / /_/ / /_/ /
/_/ /_|\___/_/_/\____/    /_/  \____/_.___/\__, /
                                          /____/
```

評價：向右斜 45 度，動感強烈，適合強調重要標語。

### 3. 迷你字體（mini）

```bash
$ figlet -f mini "Hello Toby"
            __        _
|_| _ || _   | _ |_    _
| |(/_||(_)  |(_)|_)\/
                 /
```

評價：極簡壓縮高度，適合空間有限的輸出（如 email subject、log 標題）。

### 4. 氣泡字體（bubble）

```bash
$ figlet -f bubble "Hello Toby"
  _   _   _   _   _     _   _   _   _
 / \ / \ / \ / \ / \   / \ / \ / \ / \
( H | e | l | l | o ) ( T | o | b | y )
 \\_/ \\_/ \\_/ \\_/ \\_/   \\_/ \\_/ \\_/ \\_/
```

評價：每個字母外框包著，看起來像卡通對話框。很可愛，但佔用空間大。

```bash
                                    _|
                                _|_|
```

評價：方塊堆疊感，有點像 Minecraft 文字效果。適合大型標題展示。

### 6. 數位字體（digital）

```bash
$ figlet -f digital "Toby Wu"
+-+-+-+-+ +-+-+
|T|o|b|y| |W|u|
+-+-+-+-+ +-+-+
```

評價：七段顯示器風格，適合數字、時鐘、機台上常見的 LED 顯示效果。

---

### 9. 組合測試：figlet + lolcat

```bash
$ figlet "HERMES" | lolcat -p 1.5
$ figlet "TOBY" | lolcat -a -d 10
```

把 figlet 的輸出 pipe 給 lolcat，就變成彩虹顏色的 ASCII 藝術。`-p` 調整彩虹相位，`-a` 是動畫模式，`-d` 是延遲。

---

### 10. 圖案字體展示（banner）

```bash
$ figlet -f banner "OK"
 #    # ####### #     #  #####
 #    # #       ##    # #     #
 ###### #####   # #   # #     #
 #    # #       #  #  # #     #
 #    # #       #   # # #     #
 #    # ####### #     #  #####
```

`banner` 字體輸出每個字母時，筆畫邊緣有細微的裝飾效果，比較裝飾性。

## 💡 想法與心得

### 這個專案的價值

**實用性極高：** figlet 是少數「幾乎每個工程師都會用到」的 CLI 工具之一。它不需要任何設定，沒有網路依賴，輸出直接可讀，適合放進：
- `~/.bashrc` 開機訊息
- deploy script 的階段標題
- log 檔案的章節分隔
- README 裡的視覺化標題

**可擴展性很強：** 只需要把 .flf 檔案放進 `/usr/share/figlet/` 或 `~/.figlet/` 目錄，就能新增任何字體。網路上有數百種 community 製作的字體可以下載。

**缺點：**
- 無法正確處理中文、日文等雙寬度字元（Unicode 支援有限）
- 輸出是純文字，無法嵌入彩色（需要另接 lolcat）
- ASCII art 在手機上顯示效果差

---

### 字體選擇建議

| 場合 | 推薦字體 |
|------|----------|
| 一般用途 | `standard` |
| 大標題展示 | `banner` 或 `block` |
| 節省空間 | `mini` |
| 數字/時鐘 | `digital` |
| 斜體動感 | `slant` |
| 可愛風格 | `bubble` |
| 彩色輸出 | `shadow` + `lolcat` |
